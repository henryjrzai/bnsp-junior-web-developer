
# CRUD - Pegawai
Aplikasi Web Pencatatan Data Pegawai

Aplikasi ini adalah platform web yang dirancang untuk mempermudah pencatatan data pegawai dalam sebuah organisasi. Dengan menggunakan teknologi HTML, CSS, dan JavaScript untuk antarmuka pengguna serta PHP dan MySQL untuk backend dan penyimpanan data, aplikasi ini memungkinkan pengguna untuk menambahkan, mengedit, menghapus, dan melihat data pegawai secara efisien.

Fitur Utama:
1. **Penambahan Data Pegawai:** Pengguna dapat menambahkan data pegawai baru, termasuk informasi pribadi dan profesional.
2. **Pengeditan Data Pegawai:** Data pegawai yang sudah ada dapat diperbarui atau diubah sesuai kebutuhan.
3. **Penghapusan Data Pegawai:** Data pegawai yang tidak diperlukan lagi dapat dihapus dari sistem.
4. **Tampilan Data Pegawai:** Pengguna dapat melihat daftar pegawai dengan informasi yang terstruktur dan mudah dibaca.

Teknologi yang Digunakan:
- **HTML & CSS:** Untuk struktur dan gaya antarmuka pengguna.
- **JavaScript:** Untuk interaksi dinamis dan responsif pada antarmuka pengguna.
- **PHP Laravel:** Untuk logika backend dan pemrosesan data.
- **MySQL:** Untuk penyimpanan data pegawai yang aman dan terorganisir.

Aplikasi ini dirancang untuk memberikan pengalaman pengguna yang intuitif dan efisien dalam mengelola data pegawai, dengan antarmuka yang mudah digunakan dan fungsi backend yang kuat dan aman.
## Instalasi

install menggunakan composer

```bash
  composer install
```

migrasi database dan melakukan seeding
```bash
  php artisan migrate --seed
```

jalankan npm
```bash
  npm run dev
```

### Login
untuk mengakses dashboard dapat melakukan dengan authentikasi menggunakan email dan password yang dapat dilihat pada tabel user didalam database.
